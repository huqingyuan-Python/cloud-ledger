# 📒 クラウドガラス帳簿

**Language / 语言 / 言語:** [简体中文](./README.md) · [English](./README-en.md) · [日本語](./README-ja.md)

---

Flask + HTML/CSS/JS で構築されたローカルのクラウド家計簿ウェブアプリです。データはローカルの JSON ファイルに保存され、同じ LAN 内のデバイスからアクセス可能。どこでも気軽に记账できます。

> 個人や小チームに最適。データは完全にプライベートで、サードパーティのサーバーにアップロードされません。

---

## ✨ 機能

- 📱 **モバイルファースト** — スマートフォンのブラウザで直接開ける、美しいガラスモーフィズム UI
- 📊 **グラフ可視化** — 残高推移折れ線グラフ、收支占比円グラフ、カテゴリ棒グラフ
- 🎯 **貯蓄目標** — 目標を設定し、リアルタイムで進捗を表示
- 💸 **月間予算** — 月間の支出上限を設定超過時に自動警告
- 📅 **月次比較** — 上月同期との收支を比較
- 🔄 **定期支出マーク** — 家賃、光熱費など繰り返し发生的出費をマーク
- 🔍 **検索・フィルター** — 備考やカテゴリ名で記録をすばやく絞り込み
- 💾 **ローカル保存** — 全データをローカルの JSON ファイルに保存、完全にプライベート
- 📤 **バックアップ** — ワンクリックで Excel と JSON バックアップをエクスポート
- 📥 **復元** — JSON バックアップのインポート対応（差分マージ、上書きなし）

## 📦 インストール

### 動作環境

- Python 3.8+
- Flask

### セットアップ

```bash
# 1. このプロジェクトをクローンまたはダウンロード
git clone https://github.com/huqingyuan-Python/CyberLedger.git
cd CyberLedger

# 2. 依存関係をインストール
pip install flask

# 3. サーバーを起動
python main.py
```

---

## 🚀 使い方

起動後、ターミナルにアクセス地址が表示されます：

```
╔══════════════════════════════════════════════════╗
║  クラウドガラス帳簿 v1.0                      ║
╠══════════════════════════════════════════════════╣
║  📍 ローカル:     http://127.0.0.1:5000        ║
║  📱 LAN:         http://192.168.x.x:5000      ║
╚══════════════════════════════════════════════════╝
```

- **ローカルアクセス**: パソコン浏览器で直接開く
- **LANアクセス**: 手机とパソコンが同じ WiFi に接続されていることを確認しブラウザで LAN 地址を開く

### コマンドライン引数

```bash
python main.py -p 8080        # ポートを 8080 に設定
python main.py --debug        # デバッグモードを有効化（コード変更時に自動再読み込み）
python main.py -p 3000 --debug
```
