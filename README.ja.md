# CyberLedger

[English](./README.en.md) | [简体中文](./README.zh-CN.md) | [繁體中文](./README.zh-TW.md) | [日本語](./README.ja.md)

---

ローカルクラウド家計簿システム。スマホとパソコンが同じLAN内有れば使える、データはすべてローカルに保存、第三方サーバーには送信しません。

---

## こんな人におすすめ

- Alipayや微信支付に財務データを渡したくないが、どこでも確認したい
- たまにスマートフォンで支出を記録して、PCで統計を見たい
- チームで簡単な経費管理がしたい

---

## 主な機能

### 家計簿の基本

- 収入・支出を記録、カテゴリ設定（食費、交通費、買い物、娱乐など）
- 各記録にメモ追加、定期的な支出（家賃，光熱費など）にマーク
- 日付範囲でフィルター、メモやカテゴリでキーワード検索

### グラフ表示

- 残高推移折れ線グラフ：1ヶ月の錢の動きが見える
- 収入支出 円グラフ：錢がどこ去了かが一目でわかる
- カテゴリ棒グラフ：各カテゴリ支出の比較

### 目標管理

- 月間支出予算設定、超過すると通知
- 貯蓄目標設定、今の進捗をパーセンテージで表示

### データ管理

- Excel出力対応、PCで表計算や分析が可能
- JSONバックアップ出力、データ消失防止
- JSONバックアップ取込対応、增量マージ（上書きしない）

## インストール方法

### 1. Python の準備

Python 3.8 以上が必要です。ターミナルで python --version を入力して確認。未インストールの場合は python.org からダウンロード。

### 2. プロジェクトのダウンロード

```bash
git clone https://github.com/huqingyuan-Python/CyberLedger.git
cd CyberLedger
```

git を使えない場合は、GitHub ページの Code ボタンから Download ZIP を選択して解凍してください。

### 3. 依存ライブラリのインストール

```bash
pip install flask flask-cors colorama
```

他のPythonプロジェクトと冲突する場合は、仮想環境を作成してください：

```bash
# 仮想環境を作成（1回だけ）
python -m venv .venv

# 仮想環境を有効化
# Windows:
.venv\Scripts\activate
# Mac/Linux:
source .venv/bin/activate

# 依存ライブラリをインストール
pip install flask flask-cors colorama
```

### 4. 実行

```bash
python main.py
```

ターミナルにアクセス地址が表示されます：

```
╔══════════════════════════════════════════════════╗
║           CyberLedger v1.0                       ║
╠══════════════════════════════════════════════════╣
║  Local:   http://127.0.0.1:5000                  ║

```

## アクセス方法

### PC からアクセス

ブラウザで http://127.0.0.1:5000 または http://localhost:5000 を開いてください。

### スマホからアクセス（LAN）

1. スマホとPCが同じWiFiに接続されていることを確認
2. ターミナルに表示された Network のアドレスを確認
3. スマホのブラウザでアクセス

> スマホからアクセスできない場合は、ファイアウォール設定を確認するか、ポートを変更してみてください。

### コマンドラインパラメータ

```bash
# ポート指定（デフォルト 5000）
python main.py -p 8080

# デバッグモード有効（コード変更時に自動再読み込み）
python main.py --debug

# 組み合わせ
python main.py -p 3000 --debug
```
